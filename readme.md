La presente repository è il modulo root che contiene i seguenti submoduli
- [prjs/mpfw](https://github.com/MuraDaco/mpfw/tree/main): modulo di progetto in cui sono presenti il codice sorgente e la documentazione del progetto **MPFW / FW2**; lo scopo del progetto è quello di implementare un ambiente di sviluppo (moduli **mpfw_---**) e un insieme di librerie (moduli **mpfw_code_libs_fw2_---**) per applicazioni embedded **multipiattaforma** su microprocessori 8, 16 e 32 bit 
- [gmms](https://github.com/MuraDaco/gmms/tree/main): modulo di utility per una gestione veloce dei submodules di git di cui è costituito il modulo  [mpfw](https://github.com/MuraDaco/mpfw/tree/main).
